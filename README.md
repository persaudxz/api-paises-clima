# 🌍 API de Países - Spring Boot

Projeto desenvolvido para consumir a API REST Countries.

## 🚀 Funcionalidades
- Buscar país pelo nome
- Exibir:
  - Nome
  - Capital
  - Região
  - População
  - Bandeira

## 🛠️ Tecnologias
- Java
- Spring Boot
- Thymeleaf
- REST API

## ▶️ Como rodar
1. Abrir o projeto
2. Rodar a classe `PaisesApplication`
3. Acessar: http://localhost:8080
